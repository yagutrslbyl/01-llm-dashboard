What is an LLM?

LLM stands for Large Language Model.

Large Language Model, is a type of artificial intelligence model that has been trained on vast amounts of text data to understand and generate human-like language. These models are designed to process and analyze natural language, allowing them to perform a wide range of tasks such as language translation, text summarization, question answering, and even creative writing.

An LLM is a massive neural network trained on huge amounts of text to predict the next token using transformer-based attention mechanisms.

  • PaLM

What is a token?

In Large Language Models (LLMs), a token is a small chunk of text that the model reads and processes. A token is not always a full word. On average, one token is roughly 3–4 characters or about ¾ of a word in English.

LLMs do not read sentences the same way humans do. Instead, they break text into tokens before processing it. These tokens can be words, parts of words, punctuation marks, or even spaces.

Tokens are important because they are the unit used to measure both context windows and pricing. The context window determines how much information the model can remember in a conversation, while API costs are usually calculated based on the number of input and output tokens.

Different languages consume tokens differently. English is usually more token-efficient because many English words are common in the model’s vocabulary. Azerbaijani words are often longer and more complex, so they may be split into more tokens.

For example, the sentence: "Hello, world!" might be split into tokens like: ["Hello", ",", " world", "!"]
